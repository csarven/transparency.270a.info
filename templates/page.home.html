Transparency International Linked Data

This site contains Transparency International data published using the Linked Data design principles. There is an in-depth explanation about this site. Or check these out to get a quick feel for what's here and some of the things you can find out:

  • O Canada! (links to DBpedia, Eurostat Linked Data, Geonames, and World Bank Linked Data).
  • Transparency International datasets (Corruption Perceptions Index and original Sources).
  • Query result for Corruption Perceptions Index 2011 country ranking
  • Switzerland ranked 5th (2009), 8th (2010), 8th (2011). You can find out the Corruption Perceptions Index country ranking by simply replacing CH code (which is the code for Switzerland) with the country code you want (take a look at the country list), and one of the years from 2009, 2010, or 2011 e.g., Canada's ranking in 2010.
